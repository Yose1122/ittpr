ITTPR TERMS

COMMON PACKAGE RULES

These rules apply to Basic, Standard, and Full Vibe.

EVERY PACKAGE INCLUDES

  • Design for one profile only.
  • Personal use for the client.
  • Adding and arranging the client’s social links.
  • Making the design work on desktop and mobile as well as reasonably possible.
  • The client must provide all images, text, and links ready to use.
  • Final delivery as complete code with short installation instructions.

NO PACKAGE INCLUDES

  • Designing a logo for the client.
  • Creating, illustrating, or drawing images from scratch.
  • Writing long-form content instead of the client.
  • Building a complete multi-page website.
  • Building an online store, login system, or database.
  • Completely changing the design concept after work has started.
  • Reselling, redistributing, sharing, or publishing the delivered code as a template.
